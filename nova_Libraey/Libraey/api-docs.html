Introdução

Bem-vindo à documentação da API Libraey. Esta API foi desenvolvida para fornecer recursos de gerenciamento de biblioteca, recomendações de livros baseadas em estados emocionais e integrações com IA para a plataforma Social Library.

O servidor da API é inicializado na porta 4567 por padrão.

Livros

Endpoints para gerenciar o catálogo de livros.

GET /api/books

Retorna todos os livros disponíveis no catálogo.

Exemplo de Resposta:

[ { "id": "dom-casmurro", "title": "dom casmurro", "content": "Uma noite destas, vindo da cidade para o Engenho Novo, encontrei no trem da Central um rapaz aqui do bairro, que eu conheço de vista e de chapéu..." }, { "id": "o-cortico", "title": "o cortiço", "content": "João Romão foi, dos treze aos vinte e cinco anos, empregado de um vendeiro que enriqueceu entre as quatro paredes de uma suja e obscura taverna nos refolhos do bairro do Botafogo..." } ]
GET /api/books/:id

Retorna um livro específico pelo ID.

Parâmetros:

Nome Tipo Descrição
id String ID único do livro

Exemplo de Resposta:

{ "id": "dom-casmurro", "title": "dom casmurro", "content": "Uma noite destas, vindo da cidade para o Engenho Novo, encontrei no trem da Central um rapaz aqui do bairro, que eu conheço de vista e de chapéu..." }
POST /api/books

Adiciona um novo livro ao catálogo.

Corpo da Requisição:

{ "title": "Grande Sertão: Veredas", "content": "Nonada. Tiros que o senhor ouviu foram de briga de homem não, Deus esteja..." }

Exemplo de Resposta:

{ "message": "Book added successfully" }
PUT /api/books/:id

Atualiza um livro existente.

Parâmetros:

Nome Tipo Descrição
id String ID único do livro

Corpo da Requisição:

{ "title": "Dom Casmurro - Edição Atualizada", "content": "Uma noite destas, vindo da cidade para o Engenho Novo..." }

Exemplo de Resposta:

{ "message": "Book updated successfully" }
DELETE /api/books/:id

Remove um livro do catálogo.

Parâmetros:

Nome Tipo Descrição
id String ID único do livro

Exemplo de Resposta:

{ "message": "Book deleted successfully" }

Livros Infantis

Endpoints para gerenciar livros infantis.

GET /api/kidsbooks

Retorna todos os livros infantis disponíveis.

Exemplo de Resposta:

[ { "id": "alfabeto-aventura", "title": "A Aventura do Alfabeto", "content": "Ana estava animada para começar sua aventura com as letras...", "ageRange": "4-6", "quizQuestions": [ { "question": "Qual foi a primeira letra que Ana encontrou?", "options": ["A", "B", "C", "D"], "correctAnswer": 0 } ] } ]
GET /api/kidsbooks/:id

Retorna um livro infantil específico pelo ID.

Parâmetros:

Nome Tipo Descrição
id String ID único do livro infantil
POST /api/kidsbooks

Adiciona um novo livro infantil.

Corpo da Requisição:

{ "title": "Os Ninjas dos Números", "content": "No mundo secreto da Matemática, existem ninjas especiais...", "ageRange": "7-9", "quizQuestions": [ { "question": "Qual é o resultado de 5 + 3?", "options": ["7", "8", "9", "10"], "correctAnswer": 1 } ] }

Recomendações de Livros

Endpoints para gerenciar recomendações de livros baseadas em estados emocionais.

GET /api/recommendations

Retorna todos os estados emocionais e suas recomendações.

Exemplo de Resposta:

{ "anxiety": [ "O Poder do Agora - Eckhart Tolle", "Respire - James Nestor", "Ansiedade: Como enfrentar o mal do século - Augusto Cury" ], "depression": [ "O Demônio do Meio-Dia - Andrew Solomon", "Perto do Coração Selvagem - Clarice Lispector", "Razão e Sensibilidade - Jane Austen" ] }
GET /api/recommendations/:state

Retorna recomendações para um estado emocional específico.

Parâmetros:

Nome Tipo Descrição
state String Estado emocional (ex: anxiety, depression, positive)
POST /api/recommendations/:state

Adiciona uma recomendação a um estado emocional.

Parâmetros:

Nome Tipo Descrição
state String Estado emocional (ex: anxiety, depression, positive)

Corpo da Requisição:

{ "recommendation": "A Coragem de Ser Imperfeito - Brené Brown" }

Idiomas

Endpoints para gerenciar idiomas disponíveis.

GET /api/languages

Retorna todos os idiomas disponíveis.

GET /api/languages/:id

Retorna um idioma específico pelo ID.

POST /api/languages

Adiciona um novo idioma.

Corpo da Requisição:

{ "name": "Espanhol", "code": "es", "flag": "🇪🇸", "vocabularyItems": [ { "word": "Libro", "translation": "Livro" }, { "word": "Hola", "translation": "Olá" } ] }

Inteligência Artificial (Gemini API)

Endpoints para interações com a API Gemini do Google.

POST /api/ai/analyze-sentiment

Analisa o sentimento de um texto.

Corpo da Requisição:

{ "text": "Estou me sentindo muito triste e desmotivado ultimamente." }

Exemplo de Resposta:

{ "emotionalState": "depression", "analysis": "O texto indica sinais de depressão, com sentimentos de tristeza e desmotivação persistentes." }
POST /api/ai/chat

Interage com o chatbot baseado em IA.

Corpo da Requisição:

{ "message": "Pode me recomendar alguns livros para ansiedade?" }
POST /api/ai/recommend

Obtém recomendações personalizadas de livros baseadas em um estado emocional.

Corpo da Requisição:

{ "emotionalState": "anxiety", "additionalInfo": "Também gosto de ficção científica." }
